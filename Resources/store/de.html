View Snapshots ist ein Plugin das es erlaubt Seitenaufrufe in Form von reinen Daten (URL Parametern, Template-Variablen, verdendetem Template, usw) als Momentaufnahme zu speichern und in der Folge unbegrenzt oft und weitestgehend identisch zur reproduzieren.

Ziel

Die aufgenommenen Snapshots können über eine Backend-App aufgerufen und in ursprünglicher Form betrachtet werden. Dadurch können Anpassungen am Template, CSS oder JavaScript unter vorher aufgezeichneten Zuständen getestet werden, bspw. eine Kombination aus mehreren Produkten im Warenkorb.

Features

  • Start/stop der Aufnahme einer Session zu jeder Zeit
  • Backend Anwendung zum Ansehen und Verwalten aufgenommener Sessions
  • Wiederherstellung aufgenommener Snapshots
  • Schritt vorwärts/zurück Funktion innerhalb mehrerer Aufnahmen einer Session
  • Vergleich der Rohdaten von Aufnahmen in Form von Diffs

Nutzung

Nach der Installation finden Sie die Backend Anwendung unter dem Menüpunkt Einstellungen.

Die Backend Anwendung zeigt alle aufgenommenen Sessions.

Um damit zu beginnen, eine Session aufzunehmen, verwenden Sie die JavaScript Konsole innerhalb der Entwicklerwerkezeuge Ihres Browsers und verwenden Sie dort folgende Befehle:

Aufname starten 
    > snapshots.record()
    ▶️️ Recording of session starting next request. Session ID: 0sorspi2md1j2vpt9tn9po3om1
Aufnahme stoppen 
    > snapshots.stop()
    ✋️️ Stopped recording current session.

Beim Abspielen aufgenommener Sessions

Springe zu nächstem Snappshot innerhalb aufgenommener Session 
    > snapshots.next()
Springe zu vorgherigem Snapshot 
    > snapshots.prev()
